# 🔧 Настройка Google Drive API

Для включения облачной синхронизации истории чатов нужно настроить Google Drive API.

## 📋 Шаги настройки

### 1. Создание проекта в Google Cloud Console

1. Перейди на https://console.cloud.google.com/
2. Нажми "Создать проект" или выбери существующий
3. Дай проекту название, например "SchoolAI Platform"

### 2. Включение Google Drive API

1. В меню слева выбери "API и сервисы" → "Библиотека"
2. Найди "Google Drive API"
3. Нажми "Включить"

### 3. Создание учетных данных

1. Перейди в "API и сервисы" → "Учетные данные"
2. Нажми "+ Создать учетные данные"
3. Выбери "ID клиента OAuth"
4. Настрой экран согласия (если требуется):
   - Тип приложения: External
   - Название: SchoolAI Platform
   - Email: твой email
   - Добавь области: `.../auth/drive.file`
5. Выбери тип приложения: "Классическое приложение"
6. Дай название: "SchoolAI Desktop"
7. Нажми "Создать"

### 4. Скачивание credentials.json

1. После создания нажми кнопку "Скачать JSON"
2. Переименуй файл в `credentials.json`
3. Положи файл в корень проекта (туда же где main.py)

## 🚀 Использование

После настройки запусти программу:

```bash
python main.py
```

При первом использовании:
1. Программа откроет браузер для авторизации
2. Войди в свой Google аккаунт
3. Разреши доступ к Google Drive
4. После этого создастся файл `token.pickle` для автоматической авторизации

## 📁 Структура файлов

```
school-ai-platform/
├── credentials.json        # Скачанные credentials (НЕ коммитить!)
├── token.pickle           # Токен авторизации (создаётся автоматически)
├── main.py                # Главный файл
└── google_drive_sync.py   # Модуль синхронизации
```

## 🔍 Проверка работы

После настройки:
1. Запусти `python main.py`
2. Выбери "y" когда спросит про облачную синхронизацию
3. Если всё настроено правильно, увидишь:
   ```
   ✅ Подключено к Google Drive
   📁 Найдена папка: SchoolAI_ChatHistory
   ☁️ Облачная синхронизация включена
   ```

## 🐛 Возможные проблемы

### Ошибка: "credentials.json не найден"
- Убедись что файл `credentials.json` лежит в корне проекта
- Проверь что он называется именно `credentials.json`

### Ошибка при авторизации
- Проверь что Google Drive API включен в консоли
- Убедись что добавил правильные области доступа
- Попробуй удалить `token.pickle` и авторизоваться заново

### Не появляется папка в Google Drive
- Проверь права доступа в настройках OAuth
- Папка создаётся автоматически при первом использовании

## 📝 Альтернативы

Если не хочешь настраивать Google Drive API, можешь использовать:

1. **Локальное хранение** (по умолчанию)
   - История сохраняется в папке `chat_history/`
   - Работает без настройки

2. **Dropbox** (требует доработки)
   - Можно добавить модуль для Dropbox API

3. **Firebase** (требует доработки)
   - Для реал-тайм синхронизации
